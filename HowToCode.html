Fun with Java

At this moment of time, I have only taken about 3 courses. One of which was a Java beginner course by Bryson Payne. The course was fun, taught me the basic fundamentals about programming such as Variables, loops all whilst making some cool good stuff.

The course was laid out with three different projects, a guess the number 1 to 100 game, secret message, and a bubble draw app. Also for each of these projects, we made three different versions where we make a console, graphical user interface(GUI) and an Android app version. So the course was a bit repetitive in that sense. I haven't actually finished the course, but I felt I learned enough to start making my own projects.
